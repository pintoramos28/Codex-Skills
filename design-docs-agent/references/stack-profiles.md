# Stack Profiles

Define a small set of default stack profiles. Select one based on repo context.

## Profile: Node + React + Postgres (default)
- Frontend: React + Vite
- Backend: Node.js + Express or Fastify
- API: REST
- Database: Postgres
- Infra: Docker + CI

## Profile: Next.js Fullstack
- Frontend/Backend: Next.js
- API: REST or tRPC
- Database: Postgres
- Infra: Vercel or containerized

## Profile: Python + FastAPI
- Backend: FastAPI
- API: REST
- Database: Postgres
- Infra: Docker + CI
